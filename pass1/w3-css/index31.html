HTML ຍໍ້ມາຈາກ Hyper Text Markup Language ເປັນພາສາຂຽນເວັບ ໃຫ້ໜ້າເວັບສະແດງ ຊີ່ງເປັນພາສາຫຼັງທີ່ຕ້ອງໃຊ້ຂຽນເວັບ

CSS ຍໍ້ມາຈາກ Cascading Style Sheets ແມ່ນພາສາທີ່ຈັດຮູບແບບຂອງເອກະສານ HTML ໃຫ້ມີຄວາມງາມຫຼາຍຂື້ນ ເຮັດໃຫ້ເວັບພັດທະນາໄວຂື້ນ ແລະ ມັນສາມາດຄວບຄຸມຮູບແບບຂອງຫນ້າເວັບຕ່າງໆໃນເວລາດຽວກັນ
